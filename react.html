React JS

The hot new framework from Facebook

Pros and Cons

Pros

  1. Blindly re-render without worrying about DOM thrashing,
  2. Their virtual DOM also resolves issues with eventing across browsers by abstracting it to a standards-compliant event-emitting/bubbling model.
  3. It's just a view layer, not a complete framework.

Cons

  1. The template syntax and the way you create DOM (with JSX) is a bit odd
  2. If you want really finite and explicit control over how things get applied to the DOM you don't really have it anymore.
  3. While you can just re-render the entire react view, depending on the complexity of component, it sure seems like there can be a lot of diffing to do.
