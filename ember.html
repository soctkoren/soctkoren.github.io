Pros and Cons

Pros

  1. Heavy emphasis on doing things "The Ember Way"
  2. Outsource many of the hard problems of building single page apps
  3. Big, helpful community.
  4. Nice docs site.
  5. A good amount of existing solved problems and components to use.

Cons

  1. Heavy emphasis on doing things "The Ember Way".
  2. Ember codifies a lot of opinions.
  3. Memory usage can be a bit of an issue, which can be a problem, especially when running Ember on mobile
  4. Ember is intentionally, and structurally inflexible.
